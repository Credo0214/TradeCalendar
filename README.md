# TradeCalendar

**TradeCalendar** は、  
トレードにおける **資金管理と振り返りを安全に行うための iOS アプリ**です。

ロット計算や複雑な数式を排除し、  
**「このトレードで最大いくら失ってよいか（1R）」**を  
常に可視化することに特化しています。

---

## 🎯 コンセプト

- 判断ミスを減らす
- 感情的なトレードを抑制する
- 実運用で事故を起こさない

**計算よりも「意思決定の補助」に重点を置いた設計**です。

---

## ✨ 主な機能

### 1. リスク管理（中核機能）
- 許容損失額（1R）を自動算出
- 2R / 3R を目標値として表示
- すべて **表示専用（DB保存なし）**

**数式**
許容損失額 = 資金 ×（リスク％ / 100）


---

### 2. トレード記録
- 日付・通貨ペア・損益・メモを記録
- カレンダー形式で一覧表示
- 日次・月次の損益を自動集計

---

### 3. 損益グラフ
- 累積損益の推移を可視化
- 期間切り替え（1ヶ月 / 3ヶ月 / 全期間）
- 最大ドローダウン表示
- ドラッグで日付・損益を確認

---

### 4. 設定
- リスク率（％）を一元管理
- ダークモード / ライトモード対応
- 設定は **Single Source of Truth** で管理

---

## 🧠 設計思想（重要）

### 意図的に「やらないこと」
- ロット計算
- pip / tick / レバレッジ計算
- 勝率・期待値の自動算出

👉 **誤用や過信による事故を防ぐため**です。

---

### リスク管理の考え方
- 1R = 「このトレードで最大いくら失ってよいか」
- 利益は 2R / 3R を目安に考える
- すべて **意思決定支援のための表示**
